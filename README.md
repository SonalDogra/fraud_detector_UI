# Fraud Detector UI (Spring Boot)

A frontend-facing Spring Boot application that acts as a **prototype UI**
for a fraud detection system. It allows users to submit transaction and
behavioral parameters to a backend fraud detection engine for analysis.

This project is designed as the **UI and API gateway layer** for a larger
fraud detection architecture.

## Features
- User interface for submitting transaction details
- REST API endpoints for fraud detection requests
- Clean Spring Boot project structure
- Dockerized application for easy deployment
- Ready to integrate with ML / AI-based fraud detection services

## Tech Stack
- Backend / UI Layer: Spring Boot
- Language: Java
- Build Tool: Maven
- Containerization: Docker
